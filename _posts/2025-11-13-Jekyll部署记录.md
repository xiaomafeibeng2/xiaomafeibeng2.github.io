首先梳理下生成文章的大致流程。在win的机器上部署好jekyll后，将Typora编辑好的文章放在_posts目录下（实际上可以直接使用Typora打开这个目录编辑），然后使用git提交,就可以在GitHub page看到文章，非常方便。

#### 部署Jekyll

1、安装ruby,在win环境下我们使用的是RubyInstaller

```
https://rubyinstaller.org/
```

2、然后使用gem命令安装jekyll

```
gem install jekyll bundler
```

3、创建一个站点

```
jekyll new my-awesome-site
```

4、进入创建的站点文件夹

```
cd my-awesome-site
```

5、安装依赖

```
bundle install
```

6、启动项目

```
bundle exec jekyll serve
```



#### 创建github page

1、创建一个项目，repository name必须为<font color='red'>用户名.github.io</font>

2、使用git，将之前的项目目录commit到这个repository 

```
# 初始化Git仓库
git init
```

```
# 设置用户名（替换为你的GitHub用户名）
git config --global user.name "Your GitHub Username"

# 设置邮箱（替换为你的GitHub注册邮箱）
git config --global user.email "your.email@example.com"
```

```
# 连接到GitHub远程仓库（可能会跳出web登录页面，填写github账户密码登录就行）
git remote add origin https://github.com/your-username/your-repository.git
```

```
# 添加所有文件到暂存区
git add .

# 提交更改
git commit -m "Initial commit"

# 推送到GitHub
git push -u origin main
```
