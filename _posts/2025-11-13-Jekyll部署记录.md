最近尝试使用jekyll做一些记录，为了防止以后忘记，先记录下jekyll的部署过程。

首先梳理下生成文章的大致流程。在win的机器上部署好jekyll后，将Typora编辑好的文章放在_posts目录下（实际上可以直接使用Typora打开这个目录），然后使用git提交,就可以在GitHub page看到文章，非常方便。

部署的步骤大概如下

1、安装ruby,在win环境下我们使用的是RubyInstaller

```
https://rubyinstaller.org/
```

2、然后使用gem命令安装jekyll

```
gem install jekyll bundler
```

3、创建一个站点

```
jekyll new my-awesome-site
```

4、进入创建的站点文件夹

```
cd my-awesome-site
```

5、安装依赖

```
bundle install
```

6、启动项目

```
bundle exec jekyll serve
```
